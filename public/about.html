About Sportscart

Your student-built sports shopping platform.

Sportscart is an e-commerce web application created for a university web programming project. It is designed to provide a clean and user-friendly shopping experience for sports products, while also including admin management, review handling, order tracking, and dashboard features.

Our Mission

Our mission is to make sports product shopping simple, modern, and easy to manage for both customers and administrators.

What We Offer

Sportscart allows users to browse products, add items to cart, place orders, leave reviews, and track order progress through a clean dashboard.

Admin Features

The admin can manage products, update order statuses, review customer activity, and maintain overall control of the system.

Project Highlights

  • User registration and login
  • Role-based admin and customer access
  • Dynamic products from PostgreSQL
  • Shopping cart and order placement
  • Review and likes system
  • Dashboard for customer and admin

Why This Project Matters

This project demonstrates practical skills in frontend development, backend APIs, PostgreSQL database design, authentication, CRUD operations, and responsive UI design.

It reflects the structure of a real-world online store while remaining suitable for an academic project submission.
